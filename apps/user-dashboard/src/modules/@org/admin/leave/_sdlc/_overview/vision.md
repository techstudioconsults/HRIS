# Leave Management — Vision

_Empower HR admins with a unified hub to configure leave policies, manage leave types, and action employee requests with full auditability._

## Problem Statement

Organisations currently manage leave requests through disconnected spreadsheets and email chains, leading to policy inconsistencies, delayed approvals, and poor employee experience. This module replaces that with a structured, rules-driven leave management system.

## Strategic Goals

- Provide HR admins a single interface to configure all leave types (annual, sick, casual, bereavement, maternity, paternity) with per-type rules such as carry-over caps, cycle reset dates, and minimum eligibility tenure.
- Enable fast, auditable approval and decline workflows with optional delegation to line managers.
- Support first-run setup for newly onboarded organisations via a guided wizard that pre-populates sensible defaults.

## Success Metrics

- Leave request processing time reduced from days to hours.
- Zero policy-misconfiguration incidents due to enforced validation on leave type setup.
- 100% of leave decisions are logged with actor identity, timestamp, and reason for audit compliance.
