# User Payslip — Roadmap

## Phase 1 — Core Payslip View (Current)

- Paginated payslip grid (card layout), most recent first.
- Summary card with latest net pay.
- Payslip details modal with earnings, deductions, and summary figures.
- PDF download of any payslip.
- Payslip history for full employee tenure.

## Phase 2 — Enhanced UX

- Year filter to narrow payslip list by calendar year.
- Cumulative YTD (year-to-date) earnings summary.
- Email payslip to self.

## Phase 3 — Future

- Payslip dispute / query flow (HR notification).
- Tax certificate (annual IRS/FIRS summary) download.
