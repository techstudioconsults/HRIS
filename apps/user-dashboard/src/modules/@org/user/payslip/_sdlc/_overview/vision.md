# User Payslip — Product Vision

## Problem

Employees need a convenient, self-service way to review their payslips and download them for personal record-keeping or third-party use (loan applications, visa applications). Without this, employees raise HR tickets for routine payslip requests.

## Vision

A clean, read-only payslip history that employees can access at any time. The experience is simple: see your latest pay at a glance, browse history, and download any payslip as a PDF in one click.

## Success Metrics

- Zero HR tickets for "please send me my payslip" requests after launch.
- Payslip list loads in < 1.5s.
- PDF download completes in < 5s.
- All payslips accessible for the employee's full tenure.
