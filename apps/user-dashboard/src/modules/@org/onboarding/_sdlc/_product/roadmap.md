# Onboarding — Product Roadmap

- [x] Welcome page with optional Driver.js tour
- [x] Step 1 — Company profile setup
- [x] Step 2 — Teams and roles configuration
- [x] Step 3 — Employee invitation (batch)

- [ ] Guided tour for admin dashboard (separate from onboarding tour)
