# EPIC-001 — Employee Payslip Self-Service

## Goal

Employees can view and download all their payslips without raising an HR ticket.

## User Stories

| ID     | Story                                                                                  | Priority |
| ------ | -------------------------------------------------------------------------------------- | -------- |
| US-001 | As an employee, I can see my latest net pay on a summary card                          | P0       |
| US-002 | As an employee, I can browse all my payslips in a grid                                 | P0       |
| US-003 | As an employee, I can open a payslip to see the full earnings and deductions breakdown | P0       |
| US-004 | As an employee, I can download any payslip as a PDF                                    | P1       |

## Acceptance Criteria (high-level)

- Summary card always shows the most recent payslip's net pay.
- Grid is sorted most-recent-first.
- Details modal shows all earnings line items, all deductions line items, gross pay, total deductions, and net pay.
- PDF download is accessible from both the grid card and the details modal.
- All data is scoped to the authenticated employee — no cross-employee data exposure.

## Dependencies

- `admin/payroll` — payslips are generated by the admin payroll run. This module only reads them.
- `PayslipStatus` type is shared from `admin/payroll/types`.
