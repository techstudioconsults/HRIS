# Admin Teams — Product Vision

## Problem

Growing organisations need structured team management: grouping employees, defining roles with explicit permissions, and onboarding new members efficiently. Without this, permission drift and unclear reporting structures create security and operational risk.

## Vision

The Teams module provides a guided, admin-controlled workflow for building team structures. Admins create teams, define custom roles with granular permissions, and assign employees in a single coherent flow — not three disconnected screens.

## Success Metrics

- Admin can create a team, add a role, and assign an employee in under 3 minutes.
- Zero cross-organisation team data leakage (enforced by JWT `organisationId` scoping).
- Team roster export available and accurate within 10 seconds.
- All permission changes are auditable.
