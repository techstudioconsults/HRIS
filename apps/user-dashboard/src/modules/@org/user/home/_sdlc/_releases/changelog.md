# User Home — Changelog

_Release history for the employee home dashboard module._

## [Unreleased]

### Added

- Initial `_sdlc/` documentation scaffold for the home module.
- Two-variant view architecture: `ActiveUserView` and `OnboardingView`.
- `QuickActionCard` component with navigation to leave, payslip, and team views.
- `ActivityItem` component supporting all four `ActivityType` variants.
- `OnboardingHeader` component showing setup progress.
- Static `RECENT_ACTIVITIES` mock data as a placeholder for Phase 2 API integration.
- `SetupTask` types and constants (`SETUP_TASKS_TOTAL`, `SETUP_TASK_IDS`).
