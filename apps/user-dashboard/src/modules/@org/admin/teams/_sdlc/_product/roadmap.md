# Admin Teams — Roadmap

## Phase 1 — Core Team Management (Current)

- Team CRUD (create, read, update, delete)
- Role CRUD within teams with permission configuration
- Employee assignment to teams with role selection
- Guided multi-step workflow dialog (Team → Role → Employee)
- Paginated, filterable team list
- Team and sub-team detail views
- Team roster export (CSV/Excel)

## Phase 2 — Membership Controls

- Manager approval workflow for membership changes (currently `requireManagerApproval: true` is enforced by backend)
- Membership request notifications
- Bulk member assignment
- Member status lifecycle (active → inactive → removed)

## Phase 3 — Advanced Features

- Team hierarchy visualisation (org chart)
- Team budget tracking
- Team goals and OKRs
- Team analytics and performance reports
- Team chat integration
