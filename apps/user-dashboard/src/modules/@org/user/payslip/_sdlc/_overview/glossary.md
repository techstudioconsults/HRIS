# User Payslip — Glossary

| Term              | Definition                                                                                          |
| ----------------- | --------------------------------------------------------------------------------------------------- |
| **Payslip**       | A record of one pay period for an employee: gross pay, earnings breakdown, deductions, and net pay. |
| **Earnings**      | Line items that make up gross pay: base salary, allowances, bonuses.                                |
| **Deductions**    | Line items subtracted from gross: PAYE tax, pension, health insurance.                              |
| **Net Pay**       | Gross pay minus total deductions. The amount actually received by the employee.                     |
| **PayslipStatus** | Lifecycle state of the payslip. Values: `paid` (all other statuses managed by `admin/payroll`).     |
| **MonthLabel**    | Human-readable period label (e.g. "October 2025"). Used for display only.                           |
| **Summary Card**  | The card at the top of the payslip page showing the most recent net pay.                            |
| **Payslip Grid**  | The card-based list of all payslips, most recent first.                                             |
| **Details Modal** | The full-screen or large modal with the complete payslip breakdown.                                 |
