# User Home — Product Roadmap

_High-level release milestones for the employee home dashboard._

## Phase 1 — MVP (Current)

- Active-user view: personalised welcome, quick-action cards, recent activity feed.
- Onboarding view: setup checklist with progress header.
- Automatic view-switching based on `userSetupComplete` flag.

## Phase 2 — Enhancements

- Real-time activity feed: replace static mock data with live API-driven events via TanStack Query.
- Leave balance summary widget on the active-user view (shows remaining days per leave type at a glance).
- Team presence widget: show who is on leave today.

## Phase 3 — Personalisation

- Employee-configurable quick-action order (drag-and-drop card arrangement).
- Notification badge on quick-action cards (e.g., payslip available indicator).
- Announcements banner for company-wide HR notices.

## Deferred / Out of Scope

- Admin-level analytics or reporting on the employee home page.
- Third-party widget integrations (calendar, Slack status).
