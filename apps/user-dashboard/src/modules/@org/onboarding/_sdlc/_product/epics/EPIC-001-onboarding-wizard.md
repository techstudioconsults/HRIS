# EPIC-001 — Onboarding Wizard

## Goal

Get a newly registered company from "empty HRIS" to "team structure set up, employees invited, ready to operate" in a single guided session.

## User Stories

| ID     | Story                         | Status   |
| ------ | ----------------------------- | -------- |
| US-001 | Complete company profile      | ✅ Built |
| US-002 | Create teams and define roles | ✅ Built |
| US-003 | Invite founding employees     | ✅ Built |
| US-004 | Optional guided tour          | ✅ Built |
| US-005 | Setup status tracking         | ✅ Built |

## Acceptance Criteria (Epic Level)

- [ ] Owner can complete all 3 steps without leaving the browser.
- [ ] Partial completion is preserved — returning to a step pre-fills existing data.
- [ ] Each step independently saves — closing mid-step does not lose prior steps.
- [ ] Dashboard is accessible after Step 3 completes.
- [ ] Setup status reflects completion in the employee's profile.
