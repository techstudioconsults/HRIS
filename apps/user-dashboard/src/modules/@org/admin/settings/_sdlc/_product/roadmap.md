# Settings Module — Roadmap

_Phased delivery plan for the organisation settings hub._

## Phase 1 — Core Settings Tabs (Current)

**Goal**: All six settings tabs functional with read + write.

| Tab                   | Status      |
| --------------------- | ----------- |
| Account Settings      | In Progress |
| Payroll Settings      | Planned     |
| Security Settings     | Planned     |
| HR Settings           | Planned     |
| Notification Settings | Planned     |
| Roles Management      | Planned     |

## Phase 2 — Advanced Security

**Goal**: Extend Security Settings with enterprise-grade access controls.

- SSO / SAML 2.0 configuration panel.
- IP allowlist management (restrict admin access to known corporate IPs).
- Audit log viewer for security-related events (login attempts, settings changes).
- Device management (list and revoke trusted devices per user).

## Phase 3 — Payroll Rules Engine

**Goal**: Make deduction and pay rule configuration more dynamic.

- Custom deduction types (beyond the default tax/pension/health).
- Conditional deduction rules (e.g., apply health deduction only after probation period).
- Multiple pay run calendars for different employee groups (weekly contractors, monthly staff).

## Phase 4 — Notification Templates

**Goal**: Allow orgs to customise notification email content and branding.

- Editable email templates per notification event type.
- Organisation logo injected into outgoing email templates.
- Preview rendered template before saving.
