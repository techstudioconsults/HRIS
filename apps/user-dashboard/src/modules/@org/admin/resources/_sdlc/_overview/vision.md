# Resources Module — Vision

_Provide HR administrators with a centralised, organised file and folder management system for all company documents._

## Purpose

The Resources module is the single source of truth for company documentation within the HRIS. HR admins can create a meaningful folder hierarchy that mirrors their organisation's structure — for example, separating HR Policies, Employment Contracts, Onboarding Materials, and Benefits Guides into distinct top-level folders, each with nested sub-folders as needed.

## Strategic Goals

- Eliminate the need for external document storage tools by embedding resource management directly into the HRIS workflow.
- Give employees and admins a predictable, browsable folder tree instead of a flat file dump.
- Enable role-based access so sensitive documents (e.g., employment contracts, disciplinary records) are only accessible to authorised personas.

## Success Metrics

- HR admins can locate any company document within 3 clicks from the Resources landing page.
- Upload, rename, and delete operations complete with clear success/error feedback and no full-page reloads.
- All file operations are audited with `createdBy`, `updatedBy`, and timestamps persisted to the backend.
