# Resources Module — Glossary

_Domain terms and abbreviations used consistently across all Resources module documentation._

## Terms

| Term                    | Definition                                                                                                         |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **Resource**            | Any file or folder stored within the HRIS document management system.                                              |
| **Folder**              | A logical container that groups related files or sub-folders. Rendered as a `FolderCard`.                          |
| **Sub-folder**          | A folder nested inside another folder; inherits the parent's access permissions by default.                        |
| **File**                | A binary or text document uploaded by an HR admin. Rendered as a `FileCard`.                                       |
| **FolderCard**          | The UI component displaying a folder's name, icon, and contextual actions (rename, delete).                        |
| **FileCard**            | The UI component displaying a file's name, type icon, size, upload date, and actions.                              |
| **Breadcrumb**          | The navigation trail showing the current folder path, e.g. `Resources / HR Policies / 2024`.                       |
| **Folders View**        | The default tab on the Resources page; shows all top-level folders as a card grid.                                 |
| **Files View**          | The secondary tab; shows all files within the current folder scope.                                                |
| **HR Policy**           | A document defining company rules, procedures, or standards (e.g., Leave Policy, Code of Conduct).                 |
| **Onboarding Document** | Material provided to new employees during their onboarding period (e.g., Welcome Guide, Equipment Request Form).   |
| **Benefits Guide**      | Documentation explaining employee entitlements such as health insurance, pension, or wellness allowances.          |
| **Employment Contract** | A legally binding document between the employer and employee — typically restricted to HR admins only.             |
| **MIME type**           | The media type of an uploaded file (e.g., `application/pdf`, `image/png`); used for backend allowlist enforcement. |
